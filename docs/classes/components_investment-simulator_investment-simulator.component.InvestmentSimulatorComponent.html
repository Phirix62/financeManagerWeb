    Class InvestmentSimulatorComponent

    InvestmentSimulatorComponent

    This component allows users to simulate investment growth over time, given a starting amount, annual contribution, rate of return, and investment duration. It calculates yearly balances, contributions, and interest, and displays the results in a chart.
